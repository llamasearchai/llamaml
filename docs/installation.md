# Installation

## Using pip

The recommended way to install LlamaMl is via pip:

```bash
pip install llamaml
```

## From Source

You can also install LlamaMl from source:

```bash
git clone https://github.com/llamasearchai/llamaml.git
cd llamaml
pip install -e .
```

## Dependencies

LlamaMl requires:

- Python 3.8+
- Other dependencies are automatically installed by pip
